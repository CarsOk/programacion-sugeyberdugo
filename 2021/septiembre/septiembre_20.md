#### Función

```
Function misnotas(s, u, g, b, c)
    calcula = (s + u + g + b + c) / 5
    If (calcula < 7) Then
        misnotas = " Reprobó con " & calcula & " como nota final"
    Else
        misnotas = " Aprobó con " & calcula & " como nota final"
    End If
End Function
```
